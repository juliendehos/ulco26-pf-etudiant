
# spiral

## description

Permet de parcourir les coefficients d'une matrice selon différents chemins,
par exemple en spiral.

```
initial matrix:
abcdef
ghijkl
mnopqr
stuvwx

spiral:
abcdeflrxwvutsmghijkqpon

fat spiral:
abcdefghijklrxqwvutsponm

first last:
abcdefstuvwxghijklmnopqr
```
